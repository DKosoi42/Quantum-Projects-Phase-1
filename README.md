# Quantum Projects — Phase 1

A portfolio repo for my 8–10 week track on quantum information, quantum computing (Qiskit), and small quantum many-body/entanglement toy models.

## Environment
- Conda env: `qinfo`
- Create: `conda env create -f environment.yml`
- Activate: `conda activate qinfo`

## Structure
- `notebooks/` — exercises and experiments
- `src/` — reusable code
- `tests/` — unit tests (pytest)
- `figures/` — plots
